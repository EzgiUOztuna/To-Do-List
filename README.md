# TO_DO_LIST WITH REDUX

## Installations
1) React + Vite
    `npm create vite@latest`
    https://vite.dev/guide/

2) React Router
    `npm i react-router`
    https://reactrouter.com/start/declarative/installation

    Tutorial: https://www.w3schools.com/react/react_router.asp

3) Redux
    `npm i redux`

4) Redux Thunk
    `npm install redux-thunk`
